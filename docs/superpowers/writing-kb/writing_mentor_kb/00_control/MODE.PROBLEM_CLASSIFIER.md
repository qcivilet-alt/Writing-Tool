## Purpose
Classify the likely craft problem from the user's symptom language.

## Common symptom -> likely cause clusters
- "boring" -> scene purpose, tension, pacing, motive, exposition load
- "confusing" -> information release, causality gaps, POV control, world rules
- "flat character" -> weak motive, low agency, undramatized arc pressure
- "ending doesn't land" -> weak setup/payoff, climax logic, theme mismatch
- "middle drags" -> weak midpoint shift, repetitive obstacles, subplot clutter
- "dialogue feels stiff" -> low subtext, undifferentiated voice, exposition speech
- "revision broke things" -> continuity/ripple map needed
