## Purpose
Provide a Weiland-style structural map for diagnosing macro plot movement.

## Source-aligned notes
Weiland's story-structure series treats story structure as non-formulaic but highly useful for diagnosing plot problems. The core series hub groups material into major beats, linked/chiastic structure, two-halves logic, and four worlds.

## Core beat map
- Hook
- Inciting Event
- First Plot Point
- First Pinch Point
- Midpoint / Second Plot Point
- Second Pinch Point
- Third Plot Point
- Climax
- Climactic Moment
- Resolution
