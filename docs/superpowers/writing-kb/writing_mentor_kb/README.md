# Writing Mentor KB Starter Pack

This package is an operational starter corpus for a fiction-focused writing mentor skill.
It is optimized for selective retrieval, not linear reading.

## Package contents
- `00_control/`: routing and intervention controls
- `01_routing_cards/`: short dispatch cards for fast triage
- `02_leaf_guides/`: reusable craft guides with metadata
- `03_checklists/`: applied audit templates
- `04_comparative_memos/`: disagreement/crosswalk notes
- `05_dependency_maps/`: ripple-effect maps
- `06_examples/`: symptom-to-guide examples and manifest files
- `SOURCE_AUDIT.md`: what was validated against source materials and what was revised
- `BUILD_PLAN.md`: implementation plan and priority order

## Important note
This is a curated starter corpus, not an exhaustive encyclopedia.
The highest-priority revisions from the source audit were incorporated:
- Weiland-specific beat taxonomy
- Key Event vs Inciting Event vs First Plot Point
- Pinch Points
- Linked/Chiastic structure
- Two-halves major beat logic
- Four metaphoric worlds
- Climactic Moment
- Scene/sequel terminology
- Purdue pitfalls around summary vs scene, scope, stereotype risk, and overdramatic plotting
- Comparative structure lenses from Reedsy
- Revision and ripple logic from Janice Hardy search material

## Retrieval pattern
1. Detect stage
2. Classify symptom/problem
3. Choose intensity
4. Fetch 1 router
5. Fetch 1-3 leaf guides
6. Optionally fetch a checklist
7. Fetch ripple map only if a change is proposed
