## If you change protagonist motive
Recheck:
- scene goals
- dialogue tone
- midpoint meaning
- climax legitimacy
- theme argument
- relationship dynamics

## If you change a world rule
Recheck:
- prior problem-solving logic
- constraints and costs
- timeline plausibility
- antagonist capability
- exposition needs
- ending fairness

## If you change the ending
Recheck:
- foreshadowing
- thematic setup
- character arc beats
- payoff promises
- subplot closure
- emotional continuity
